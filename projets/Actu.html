À propos du projet

Ce projet est une application web dynamique développée avec le framework Symfony. Elle permet aux utilisateurs de suivre l'actualité en temps réel grâce à l'intégration de l'API GNews.

L'interface, conçue avec Bootstrap et HTML5, offre une navigation fluide. Les données utilisateurs et les favoris sont stockés via une base de données MariaDB (gérée sous MAMP), assurant une persistance fiable des informations.

Stack Technique

  • Symfony (Backend)
  • MariaDB (MAMP)
  • API GNews
  • Bootstrap 5

Fonctionnalités

  • Flux d'actualités en direct via API externe.
  • Recherche d'articles par mots-clés.
  • Authentification utilisateur sécurisée.
  • Sauvegarde des articles en favoris (Base de données).
